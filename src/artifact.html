Class Artifact.

The Artifact class is a namespace class to gather various artifact-related functions and keep them from being globals.

At the moment list() is the only function.

static string Artifact::list()

Returns a json object containing a list of installed artifacts. Format to be decided later; I don't think this is useful, so I'll just do something and if we turn out to need it, but different, we'll know how by then.
